# ** WIP **
## GOALS
- Understand the difference between passive and active data collection
- Locate Active Discovery and Active Query functionality within CTD
- Create and execute an Active Discovery task
- Create and execute an Active Query
- Understand how CTD presents information gathered through active methods
---
## TASKS
### Part One: Exploring Active Discovery
1. Navigate to Settings > Data Sources > Active Detection > Tasks.
2. Review the existing Active Discovery tasks.
3. Examine the available configuration options when creating a new task.
4. Note:
   - Task Name
   - Task Type
   - Schedule Options
   - Site Selection
   - Sensor Selection
   - Available Discovery Methods
---
### Part Two: Creating an Active Discovery Task
1. Create a new Active Discovery task.
2. Review the available task configuration options.
3. Select a site and sensor.
4. Configure the task according to the instructor's guidance.
5. Review all available scheduling options.
6. Execute the task.
7. Monitor the task status until it completes.
8. Review any results generated by the task.
---
### Part Three: Exploring Active Queries
1. Navigate to Settings > Data Sources > Active Detection > Queries.
2. Review existing queries, if present.
3. Examine the available query types.
4. Review:
   - Query configuration options
   - Scheduling options
   - Site selection options
   - Sensor selection options
---
### Part Four: Creating an Active Query
1. Create a new Active Query.
2. Select an available query type.
3. Review all available configuration settings.
4. Execute the query.
5. Monitor the query status until it completes.
6. Review the resulting information.
---
### Part Five: Comparing Passive and Active Data
1. Navigate to Visibility > Assets.
2. Select an asset identified by an Active Discovery task or Active Query.
3. Review the asset information available.
4. Compare the information gathered through active methods to the information available through passive monitoring.
5. Explore any new details that were added to the asset.
