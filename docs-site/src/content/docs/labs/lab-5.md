## GOALS
- Understand virtual zones
- Review zone information
- Associate assets with zones

## TASKS
1. Navigate to the Zones page.
2. Switch between the view options, noting how the zones are connected.
3. Create a new virtual zone:
 - You can create a new zone by selecting the + symbol while on the Zones view.
 - Name the new zone and select Add. After a moment, it will appear in the zone list.
4. Assign an asset to the new virtual zone:
 - To add assets to a zone, you will need to do so on the asset’s details page. Return to the assets list and navigate to its Overview page. You should see a header titled Virtual Zone with a blue edit symbol next to it. When you select this icon, you will see a drop down menu of all available virtual zones. Select the new zone, then press the ✔️ symbol.
