## TASKS
### Part One: Collection
1. Go to Settings > Data Sources > Interface Management.
2. Expand Advanced Network Settings and ensure all (pcap) options are selected.
3. When ready, select the record ⏺️ button under the Actions headers for one of the interfaces.
4. Enter file name (optional) and the maximum file size (default is 1MB, and the maximum is 1GB).
5. The recording will stop when the file size limit is reached, but can be stopped early by clicking the stop ⏹️ button under Actions.
6. After the recording stops, download the pcap file by selecting the Download button under Actions.
