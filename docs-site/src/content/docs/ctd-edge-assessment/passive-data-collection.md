# Verifying Passive Collection
1. Go to ``Visibility`` --> ``Assets``
1. Filter by network

Expected:
* Assets appear
* IPs match environment

# Creating Initial Inventory
Allow CTD to run passively.

Over a short period of time, the devices on the network will populate and the OT protocols will be detected.

# Identifying Gaps
Look for:
* Missing subnets
* Unknown devices
