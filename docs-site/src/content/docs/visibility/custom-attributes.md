---
## TASKS
1. Navigate to `Settings` > `Management` > `Custom Attributes`.
 - This page will list any existing custom attributes for assets or sites.
2. On the assets page, select the **(+)** icon to create a new **Custom Attribute**.
3. Create a custom attribute for *Location*.
4. Navigate to an asset page, either through the Dashboard or the sidebar menu, and select an asset. Once you select your asset, in the **Overview** or **Device Information** tab, find the **Custom attributes** header and select *+ Add a custom attribute*.
5. Add the custom  *Locatoin* attribute you created to this asset and give it an appropiate value.
